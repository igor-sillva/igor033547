# Projeto Desenvolvedor Front End

O Estado de Mato Grosso pretende oferecer um registro público de Pets e seus tutores, com uma API pública. O objetivo é avaliar a capacidade
de cadastrar, editar, excluir e apresentar os dados fornecidos pela API.

## Começando

### Primeiros passos

Clonar projeto.
```bash
 git clone https://github.com/igor-sillva/igor033547.git
```

Acessar o diretório do projeto.

```bash
cd igor033547
```
